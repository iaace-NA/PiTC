  • expand_moreGeneral Commands
    • <> indicate required parameters and [] indicate optional parameters
    • "ign" means League of Legends username
    • what does [ign resolvable] mean?
      • []: nothing needs to be specified if you want to pull up your own information
      • ign: the username (whatever it is)
      • $shortcut: any $shortcut set for a user using Lsetshortcut
      • @mention: mention any other discord user
      • ^: the summoner whose information was most recently requested
    • what does <ign list resolvable> mean?
      • ign1,ign2,ign3...ign10: comma separated list of usernames
      • ign1,$shortcut2,$shortcut3,...ign10: any mix of igns and $shortcuts, comma separated
      • lobby/champ select chat: copy and paste the chat from your premade lobby or champion select
      • post game chat: copy and paste the chat from after the game ends
    • command use is not case sensitive.

      • Permissions:
      • Server Owner
      • Server Admin:
        • Kick user
        • Manage messages
        • Ban user
        • Manage roles
        • Manage channels
      • Server Moderator:
        • Kick user
        • Manage messages
      • Bot Commander: Has a role named "bot commander"

    COMMANDS v1.5.1
    		 ACTION DEFAULT EXAMPLE 
    supportbotprefix [new prefix]
         		Changes the prefix for supportbot. If none is specified, supportbot will operate without using prefixes. 
    L
    		 supportbotprefix ! 
    Lsetting auto-opgg <on/off>
         		Toggles automatic summoner information when an op.gg link is sent. on lsetting auto-opgg off 
    Lsetting force-prefix <on/off>
         		Force using a prefix for LoL statistics command that do not require one. (mastery <-> Lmastery) off lsetting force-prefix on 
    Lsetting release-notifications <on/off>
         		Show SupportBot release notes in your server. on lsetting release-notifications off 
    supportbot settings reset all
         		Reset all server settings to default. supportbot settings reset all
